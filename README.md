# Klett Downloader

Downloader do nagrań z Chmury Klett

To jest narzędzie do pobierania nagrań audio z [Chmury Klett](https://chmura.klett.pl/) napisane w Node.js. Pozwala na pobrania wszystkich nagrań audio dla podręcznika lub ćwiczeń (bez filmów Vimeo).

## Jak uruchomić

 Wymagany jest Node.js >= 22.21.1.
 Dla użytkowników nvm:

    nvm install 22.21.1

Sklonuj repozytorium i przejdź do niego:

    git clone https://github.com/skantek/klett-downloader.git && cd klett-downloader

Zainstaluj zależności:

    npm install

Uruchom downloader:

    node index.js

Postępuj zgodnie z krokami wyświetlanymi w terminalu. Po chwili wszystko powinno zostać pobrane.
